Week 2 (Feb 04 - 08)

Goals for this week:

This week I was mainly trying to figure out how the interface of the os should work. For phones and tablets, something similar to the Android navigation buttons would be good. For PC, it is not needed, but I still haven't decided how that will be.

There would be two sections, a viewport for the applications and interface of the OS, and a navbar below it. Basically exactly like android.

All position styling rely on CSS Flexbox, which handles how items are positioned in a container.

I also added a Diary app to the app list which will contain this documentation.

The scroll position of where the user last left the Diary app is persisted so that one is able to continue reading.

The Diary app presents a list containing documentation for each week. It also maintains keeps track of which documentation is open.

I am writing the documentation with ghostwriter which is an app that exports HTML (and also handles images). The HTML files are then provided by the server for the Diary app.

The server would also provide any files that the OS applications might need in future.

Some apps will require the ability for more than once instance to be running.

The Diary app should allow multiple entries to be open, each with their own state.
